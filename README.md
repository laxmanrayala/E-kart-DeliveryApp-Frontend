# 🛒 E-Kart Frontend (React)

Frontend UI for the E-Kart application.

---

## ⚙️ Tech Stack

- React
- Tailwind CSS
- Axios
- Context API

---

## 🚀 Run Locally

npm install  
npm run dev

---

## 🌐 App URL

http://localhost:5173 (allowed cors in security config)

---

## 🔗 Backend Connection

Update API base URL:

http://localhost:8080

---

## 📦 Features

- Store selection
- Product browsing
- Add to cart
- Real-time cart UI
- Checkout flow
- Order history
- Admin dashboard

---

## 🧠 Notes

- Backend must be running before using frontend
- Make sure correct API URL is configured
